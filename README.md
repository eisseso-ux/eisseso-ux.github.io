# eisseso-ux.github.io

Personal artist website.

## Pages
- `index.html`: Home page (hero image, artist bio, contact section)
- `projects.html`: Projects + image gallery page
- `gallery.js`: Fullscreen gallery behavior (click image, next/previous arrows, close)
- `styles.css`: Site design and layout

## Easy Content Edits
- Change homepage image in `index.html` at the `hero-image` section.
- Update bio text in `index.html` under `Artist Bio`.
- Update contact details in `index.html` under `Contact Me`.
- Add/remove project images by editing `galleryData` in `gallery.js`.

## Notes
- Image paths must match files in the `images` folder.
- If a filename has spaces, keep the exact filename as written.
